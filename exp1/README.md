# Experiment 1: Student Admission Form

A simple HTML form with client-side validation for student admissions.

## Description

This experiment demonstrates a student admission form with form validation using pure HTML, CSS, and JavaScript. The form includes fields for personal information, academic details, and validates user input before submission.

## Features

- Client-side form validation
- Responsive design
- Clean and modern UI
- Real-time error messages
- Success message on valid submission

## How to Run

1. **No installation required** - This is a static HTML page

2. **Open the file**:
   - Simply open `index.html` in any modern web browser
   - OR right-click on `index.html` and select "Open with" → Your browser of choice

## Usage

1. Fill in all the required fields in the form
2. The form will validate:
   - Full Name (required)
   - Email Address (valid email format)
   - Date of Birth
   - Gender
   - Course selection
   - Address
3. Click "Submit Application" to submit the form
4. A success message will appear if all validations pass
